# Python Server-Side Rendering

This project contains progressive exercises for learning server-side rendering (SSR)
with Python, Flask, and Jinja templates.

## Tasks implemented

- Task 0: Text templating invitations generator
- Task 1: Basic Flask app with reusable header/footer templates
- Task 2: Dynamic list rendering from JSON using loops and conditions
- Task 3: Product rendering from JSON/CSV via query parameters
- Task 4: Extended product rendering with SQLite support
