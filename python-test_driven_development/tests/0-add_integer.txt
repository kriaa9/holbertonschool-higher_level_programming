The add_integer module

Using the add_integer function:
-------------------------------

>>> add_integer = __import__('0-add_integer').add_integer

Basic addition of integers:
>>> add_integer(1, 2)

Addition with negative numbers:
>>> add_integer(100, -2)

>>> add_integer(2)
100

Addition with floats (should be cast to integers):
>>> add_integer(100.3, -2)

>>> add_integer(2.7, 5.2)
