===========================
Module: 1-my_list
===========================

The ``MyList`` module defines a custom list class with sorting functionality.

>>> MyList = __import__('1-my_list').MyList

Test 1: Create an empty MyList
>>> my_list = MyList()
>>> print(my_list)
[]

Test 2: Print sorted on empty list
>>> my_list.print_sorted()
[]

Test 3: Append integers and print
>>> my_list.append(1)
>>> my_list.append(4)
>>> my_list.append(2)
>>> my_list.append(3)
>>> my_list.append(5)
>>> print(my_list)
[1, 4, 2, 3, 5]

Test 4: Print sorted (original list unchanged)
>>> my_list.print_sorted()
[1, 2, 3, 4, 5]
>>> print(my_list)
[1, 4, 2, 3, 5]

Test 5: MyList is instance of list
>>> isinstance(my_list, list)
True

Test 6: Sorted single element
>>> my_list2 = MyList()
>>> my_list2.append(42)
>>> my_list2.print_sorted()
[42]

Test 7: Already sorted list
>>> my_list3 = MyList()
>>> my_list3.append(1)
>>> my_list3.append(2)
>>> my_list3.append(3)
>>> my_list3.print_sorted()
[1, 2, 3]

Test 8: Reverse sorted list
>>> my_list4 = MyList()
>>> my_list4.append(5)
>>> my_list4.append(4)
>>> my_list4.append(3)
>>> my_list4.append(2)
>>> my_list4.append(1)
>>> my_list4.print_sorted()
[1, 2, 3, 4, 5]

Test 9: Negative numbers
>>> my_list5 = MyList()
>>> my_list5.append(-5)
>>> my_list5.append(3)
>>> my_list5.append(-1)
>>> my_list5.append(0)
>>> my_list5.print_sorted()
[-5, -1, 0, 3]

Test 10: Duplicate numbers
>>> my_list6 = MyList()
>>> my_list6.append(2)
>>> my_list6.append(1)
>>> my_list6.append(2)
>>> my_list6.append(1)
>>> my_list6.print_sorted()
[1, 1, 2, 2]

Test 11: Large numbers
>>> my_list7 = MyList()
>>> my_list7.append(1000)
>>> my_list7.append(100)
>>> my_list7.append(10)
>>> my_list7.print_sorted()
[10, 100, 1000]
